RE2 Port of 'More Ammo (or Less Ammo)' by Veoxys0910. Extensive code rewrite for RE2 but functionally works the same.

If you like the mod don't forget to endorse!

More Ammo Pickup RE2  multiplies the amount of ammo you receive from every ammo pickup in Resident Evil 2. Found 6 pistol rounds? You get 8, or 24, or 100! Want a harder playthrough? Drop it to 0.5× and get half instead. Grabbed some revolver rounds? Multiplied (or divided). Every ammo type is covered (hopefully). Only ammo is affected.

Features
Multiplies all ammo pickup quantities — defaults to 1× out of the box
Configurable multiplier via the REFramework menu — supports values from 0.0× and up, so you can boost or nerf your ammo
In-game toggle via the REFramework menu — enable or disable at any time without restarting

Requirements
Resident Evil 2
REFramework (Github)

Incompatibilities / Issues
There is a soft incompatibility with Stack Expansion Mod (RE2) mod. If you want to run both together you MUST disable the 'Pickup Convenience' feature 'Smart Ammo Pickup Merging' from that mod. If you dont then any ammo you pickup will respawn on Save/Load, it can also cause non ammo key items etc to respawn so ensure you disable that mods specific feature. Everything else from SE can stay active.

Installation
Install REFramework for RE2
Copy files into <game_dir>/reframework/
To reload without restarting: open the REFramework menu (Insert key) and use Script Generated UI → Reload All Scripts.
