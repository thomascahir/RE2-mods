Better Movement Speed. Play faster.

If you like the mod don't forget to endorse!
What It Does
Increases your walking, running, melee attack, and reload speed by a configurable multiplier.
    Default: 1.0x speed — applied automatically on install
    Independent Walk, Run, and Reload speed sliders
    Adjustable range: 0.5x ~ 3.0x
    Works in both third-person view and first-person (via REFramework)
    Customizable hotkey to toggle on/off (disabled by default)
    Toggle on/off anytime
    Settings saved automatically
Requirements
    Resident Evil 2
    REFramework (Github)
Installation
    Install REFramework for RE2
    Copy files into <game_dir>/reframework/
To reload without restarting: open the REFramework menu (Insert key) and use Script Generated UI → Reload All Scripts.
